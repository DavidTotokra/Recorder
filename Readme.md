## How to run the Audiorecorder 
1. Clone this repository  
2. RUN :

with
```cmd 
 python .\AudioRecorderTkinter.py 
```
or

```cmd 
 python .\AudioRecorderCLI.py here the link --filename=nyAudo.mp3 --duration=22
```
